1.website website is a collection of web pages,images,video and other digital content that is hosted on a web server and accessible through the internet.website created by different programming languages like html, css, javascript and server side scripting languages like php, python etc. 2.webpage a webpage is a single document or file that is displayed in a web browser when accessed through the internet. webpage written in language such as html, css,javascript. 3.web browser a web browser is a software application used to access and view web pages and other content on internet. web browser like google chrome, mozilla firefox,apple safari,opera etc. 4.web server web server is a computer system or program that is responsible for storing,processing,and delivering web content over the internet. 5.html HTML stands for Hyper Text Markup Language HTML is the standard markup language for creating Web pages HTML describes the structure of a Web page HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc. 6.css CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media
